How to make a basic website using HTML and CSS

Have you ever wanted to learn how to make a basic website? Well, this guide will show you how to get started making a website using HTML and CSS.

What is HyperText Markup Language (HTML)?

HTML is a markup coding language used to create the skeleton of a website. HTML has two main types of elements: block, and inline elements. These refer to how they will display on a website. An inline element will only take up as much room as it needs. However, a block element will take up the entire row, not allowing any elements to go beside it.

On the left, we have an inline element, and on the right, a block element.

Step 1: To start an HTML document, you must first make sure your file ends in “.html”.

You must use some HTML tags. An HTML tag, is either a word or a letter surrounded by the greater than “< ” and lesser than “>” symbols.

Step 2: Once you open your document, you must start with “< !DOCTYPE Html>” this tells the computer that this is an HTML document.

Step 3: Next, everything you will have after that will be in between the “< html> ” tags. In HTML, a lot of tags have an opening and a closing tag. The “/” represents that that tag is the closing tag.

Step 4: In between the HTML tags, we can add the “<head> </head>” tags and also the “<body> </body>” tags, everything that will display on the website, must be in between the “<body>” tags.

Step 5: We can add a title to the website by using the “<title>” tags inside of the “” and then in between the tags, we will type what we want for the title.

Step 6: Moving on to the “<body>” let's add a “<h1>” tag. This is the largest heading. There are 6 different heading in HTML “<h1>-<h6>”. The lower the number is, the bigger the heading will be. In between, the “<h1>” tags type "Hello World".

Step 7: Under the “<h1>”, add a “<p>” tag and type "My first web page" this is a paragraph tag, it lets you type in between it and it will display the text.

Basic Cascading Style Sheets (CSS)

Now let's start on Cascading Style Sheets (CSS). CSS is used to make your website look nice.

Step 8: In our head, we will add the “<style>” tags.

Step 9: Add “body { }” this lets you style everything in between the body tags.

Step 10: In between the brackets put “background-colour: black;”, “color: white;”, and “text-align: center;”. This will make your website’s background black, the text white, and the text to display in the center of the row.

Now you have managed to make your first basic website. If you want to learn more, check out "W3Schools" to learn more about HTML and CSS.
